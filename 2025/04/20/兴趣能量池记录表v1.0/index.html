🎨「兴趣能量池」记录表（高光时刻收集器）

🫧 适合存放你在兴趣活动中，突然感到闪闪发光的时刻，让它们有迹可循、有处安放。

🗓️ 日期：

（例：2025.04.17）

✨ 今天做了什么让我有“文艺奇才”感觉的事？

（例：为博客配了图，排版完成瞬间惊艳了自己）

💡 灵感爆发点：

（例：看到 Pinterest 上一个柔和蓝配白的排版，于是尝试加了渐变背景）

🎵 我当时的状态是？（多选 ✔）

记录下灵感来临时，你的状态，让它变得可复制。

  • 心情平稳
  • 音乐陪伴
  • 时间宽松
  • 没有人打扰
  • 是夜晚的静谧时间
  • 是逃避学习的空隙（笑）
  • 其他：

💖 我特别满意的细节：

（例：博客的标题字体太有感觉了 / 文案里的某句话我真喜欢）

🪴 从中得到的滋养：

（例：原来我不是“懒”，是没有触发点 / 我真的很会配图！）

🫧 如果以后灵感枯竭，可以提醒我：

（例：别逼自己创作，从看图和听音乐开始就好）

🔁 你可以这样使用它：

  • 每次完成某个「让你感觉棒极了」的作品或灵感行为就填一次；
  • 当你觉得“自己什么都不会”的时候，就翻开它；
  • 想找创作状态时，用“我当时的状态”那栏重新布置场景；
  • 它不是日记，是一池「会发光的回忆碎片」。
